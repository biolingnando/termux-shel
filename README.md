## [+] Installation & Usage
```
apt update
apt install git -y
git clone https://github.com/biolingnando/termux-shel.git
cd termux-shel
chmod +x *
sh install.sh
exit
```
### or use Single Command
```
apt update && apt install git -y && git clone https://github.com/biolingnando/termux-shel.git && cd termux-shell && chmod +x * && sh install.sh
```
